================
Code of Conduct
================

Gradient-Free-Optimizers follows the Contributor Covenant Code of Conduct to ensure
a welcoming, inclusive community for all contributors.

----

Our Commitment
--------------

We are committed to providing a friendly, safe, and welcoming environment for all
contributors, regardless of experience level, background, or identity.

----

Full Code of Conduct
---------------------

Please see our complete Code of Conduct in the repository:

`CODE_OF_CONDUCT.md <https://github.com/SimonBlanke/Gradient-Free-Optimizers/blob/master/CODE_OF_CONDUCT.md>`_

The full document covers:

- Our pledge and standards
- Expected and unacceptable behavior
- Enforcement responsibilities
- Reporting procedures
- Consequences for violations

----

Reporting Issues
----------------

If you experience or witness unacceptable behavior, please report it to:

**Email**: simon.blanke@yahoo.com

All reports will be reviewed and investigated promptly and fairly. We are committed
to maintaining confidentiality for reporters.

----

Scope
-----

This Code of Conduct applies to all project spaces, including:

- GitHub repository (issues, pull requests, discussions)
- Project documentation
- Community events
- Email communications
- Social media when representing the project

----

Summary of Standards
--------------------

**We encourage**:

- Respectful and constructive communication
- Welcoming newcomers and helping them learn
- Accepting constructive feedback gracefully
- Focusing on what's best for the community

**We do not tolerate**:

- Harassment or discriminatory behavior
- Trolling or insulting comments
- Publishing private information without permission
- Other unprofessional conduct

----

Enforcement
-----------

Community leaders will enforce these standards fairly and consistently. Violations
may result in warnings, temporary bans, or permanent removal from the project,
depending on severity and context.

----

Attribution
-----------

`Contributor Covenant <https://www.contributor-covenant.org/>`_, version 2.0.
